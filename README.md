# Employee-Back-NetCore
Backend Application using .Net Core

# Frontend Application using Vuejs
[Frontend App](https://github.com/Kreyco/employee-app-vue)

# Using App
1. Start [Backend App](https://github.com/Kreyco/Employee-Back-NetCore).
1. Start [Frontend App](https://github.com/Kreyco/employee-app-vue) with `npm run serve`.
1. Go to [App](http://localhost:8080) to use Frontend App and start checking the App.
